# AgenciaViajeApp

Edmilson Alejandro Martinez Reynosa

App de gestión de destinos turísticos.

## Funcionalidades implementadas

- Guardar destinos en Firebase Realtime Database.
- Listar los destinos guardados en la app.

> Nota: por tiempo limitado, las funcionalidades de editar y eliminar destinos **no se implementaron**, pero la base para añadirlas ya está preparada.

## Tecnologías utilizadas

- Kotlin
- Android Studio
- Firebase Realtime Database
- RecyclerView
- Glide para cargar imágenes
